Introduction

Welcome to this short exercise on inline elements in HTML. Inline elements allow you to style and emphasize small pieces of text without breaking the flow of a paragraph.
They are essential for improving readability and accessibility.

Inline elements, such as <a>, <span>, <em>, and <strong>, are used within block-level elements to modify content meaning or appearance. For instance, the word important below uses semantic emphasis:

It's important to use semantic tags like <em> and <strong> instead of non-semantic <b> or <i>, because they improve accessibility for screen readers.
